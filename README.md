<h3> 주제 : 독거노인 고독사 예방 스마트 슬리퍼</h3>

팀명/제품명 : Keepers

팀장 : 김도은

팀원 : 정준호, 조민규, 최설미, 허도형

<h3>🌼주요기능🌼</h3>

1. 로드셀 센서를 통한 무게 측정

2. 측정된 무게를 통해 생활반응 판단

3. 12시간이상 무반응시 사회복지사에게 알림
